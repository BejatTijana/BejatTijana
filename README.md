## 🚀 About Me

- 🎓 3rd-year student of **Applied Software Engineering** at the **Faculty of Technical Sciences, University of Novi Sad**
- 📍 Based in **Novi Sad, Serbia**
- 💻 Focused on **desktop application development** (WPF / MVVM) and **modern web frontends** (React + TypeScript)
- 🔌 Experience with **industrial communication protocols** (Modbus RTU/TCP), real-time systems, and distributed client-server architectures
- 🧹 Clean code enthusiast — MVVM, Services pattern, separation of concerns

---

## 💼 Currently Building

- 🔋 **DER Monitor** — real-time WPF desktop app tracking solar panels & wind turbines over TCP, with a mobile-emulated UI (390×844 px portrait mode)
- 🌤️ **Distributed Weather Service** — client-server system with CSV parsing, session management, and weather analytics
- 🎮 **JustDanceCMS** — role-based content management system for managing Just Dance characters with admin/user access levels
